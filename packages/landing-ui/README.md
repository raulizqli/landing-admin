# @raulizqli/landing-ui

Componentes React compartidos entre `landing-admin` (espejo / preview) y `landing-template` (producción).

Depende de `@raulizqli/landing-core`. Publicado en [GitHub Packages](https://github.com/raulizqli/landing-admin/packages).

El tracking de analytics se configura con `configureTracking()` — en el admin los handlers son no-op por defecto.
